### 安装指南：

如果你拥有python3环境以及该程序所需的依赖，那么恭喜你，你根本不用我说对吧，大佬您多多指点小的就好了

如果你没有......我打包了一个exe文件，在dist下，叫做SBK.exe，你用这个吧。

### 操作说明：

Alt + V 添加前景窗口到摸鱼窗口列表

Alt + B 将前景窗口设为工作窗口

Alt + N 清除摸鱼窗口列表

Alt + C 隐藏/显示摸鱼窗口，并将工作窗口最大化显示

Alt + X 退出该程序
