# SimpleBossKey  Beta 0.2.0

#### *简称[SBK](https://github.com/wza123/SimpleBossKey)，为一晚上使用python语言开发的纯快捷键交互式最小老板键软件*

~~蒟蒻第一次搞开发~~

### 安装指南：

查看Release或使用Compile.bat脚本自行编译

### 操作说明：

Alt + V 添加前景窗口到摸鱼窗口列表

Alt + B 将前景窗口设为工作窗口

Alt + N 清除摸鱼窗口列表

Alt + C 隐藏/显示摸鱼窗口，并将工作窗口最大化显示

Alt + X 退出该程序
